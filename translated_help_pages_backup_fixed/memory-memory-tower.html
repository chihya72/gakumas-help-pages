在偶像之路上，可以使用记忆所持有的参数、技能卡和P道具。

■参数
・效果
影响舞台得分奖励的参数。

■P道具
・效果
在舞台内发动各种支援效果的道具。满足条件即可发动效果。

■技能卡片
・效果
根据各计划规定的初始卡片，将成为偶像之路中使用的技能卡片。
此外，每个阶段可编组的主记忆和副记忆的上限数量是固定的。编组的记忆参数、技能卡和P道具将用于表演对决，但各自的反映对象有所不同。

■主内存
参数、技能卡、P道具将被反映。

■子存储器
部分参数和技能卡将被反映。主内存与计划类型不同的内存无法编组。
※不可重复的技能卡即使包含在多个记忆体中，也仅会反映一张。此外若包含已强化的技能卡，则优先采用已强化版本。

■参数
主内存参数和部分子内存参数将被反映。

■技能卡
编成的主/副记忆技能卡将被编入。
除了记忆持有的技能卡之外，根据主记忆的推荐效果，基本技能卡将添加到持有技能卡中。

■P道具
主内存的P项目将被编组。
