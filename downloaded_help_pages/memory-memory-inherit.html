■メモリー合成
メモリー合成チケットを使用することで、メモリーの「ステージ」のスキルカードを他のメモリーがもつ候補となるスキルカードと入れ替えることができます。
メモリーの合成には、合成元のメモリーと、合成に使うためのメモリーの2つが必要です。
※合成に使ったメモリーは、合成後消失し、なくなります
※一度合成元にして合成したメモリーは、以降メモリー合成の合成元にできなくなります

■スキルカードの入れ替え
入れ替える候補となるスキルカードは、合成に使うためのメモリーがもつスキルカードの入れ替え候補からランダムで選出されます。
※メモリー合成によって獲得できるスキルカードは、合成元の選択したスキルカードのレアリティ・強化段階より上にはなりません
※同じ名称のスキルカードを合成で獲得することはできません
※合成ではプロデュースアイドル、サポートカードがもつスキルカード、カスタマイズしたスキルカードは候補になりません
