メモリーは「プロデュース」と「コンテスト」で編成でき、それぞれで異なる効果を持ちます。
プロデュース完了時にメモリーが生成され、プロデュースの成績に応じて効果が決まります。

■プロデュース編成時
プロデュース開始時にメモリーを編成することで、プロデュース中にさまざまな効果が発生します。詳細はヘルプ項目「メモリー」の「プロデュース編成時の効果」をご覧ください。

■コンテスト編成時
コンテストでは、メモリーがもつパラメータやスキルカード、Pアイテムを使用することができます。詳細はヘルプ項目「メモリー」の「コンテスト編成時の効果」をご覧ください。

■メモリーの保護
メモリーの保護をすると、メモリーの変換の対象外にすることができます。

■メモリーのタグ
メモリーに目印としてタグを付けると、フィルタやソートでタグを参照することができます。
