アイドルへの道では、メモリーがもつパラメータやスキルカード、Pアイテムを使用することができます。

■パラメータ
・効果
ステージのスコアボーナスに影響するパラメータです。

■Pアイテム
・効果
ステージ内でさまざまなサポート効果を発動するアイテムです。条件を満たすことによって効果を発動します。

■スキルカード
・効果
プランごとに決められた初期カードとあわせて、アイドルへの道で使用するスキルカードとなります。
また、ステージごとに編成できるメインメモリーとサブメモリーの上限数は決められています。編成したメモリーのパラメータ、スキルカード、Pアイテムがパフォーマンス対決で使用されますが、それぞれ反映される対象が異なります。

■メインメモリー
パラメータ、スキルカード、Pアイテムが反映されます。

■サブメモリー
パラメータの一部と、スキルカードが反映されます。メインメモリーとプランタイプが異なるメモリーは編成できません。
※重複不可のスキルカードは、複数のメモリに含まれる場合でも、1枚のみ反映されます。また強化済みのスキルカードが含まれる場合は強化済みを優先します

■パラメータ
メインメモリーのパラメータと、サブメモリーのパラメータの一部が反映されます。

■スキルカード
編成したメイン/サブメモリーのスキルカードが編成されます。
メモリーのもつスキルカードの他、メインメモリーのおすすめ効果に応じた基本スキルカードが所持スキルカードに加わります。

■Pアイテム
メインメモリーのPアイテムが編成されます。
