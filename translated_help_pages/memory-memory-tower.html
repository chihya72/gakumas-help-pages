在偶像之路上，可以使用記憶所持有的参数、技能卡和P道具。

■参数
效果
影響踊台得分奖励的参数。

■P道具
效果
在踊台内発動各種支援效果的道具。满足条件即可発動效果。

■技能卡片
效果
根据各計画规定的初始卡片，将成为偶像之路中使用的技能卡片。
此外，每個阶段可编組的主記憶和副記憶的上限数量是固定的。编組的記憶参数、技能卡和P道具将用於表演対决，但各自的反映対象有所不同。

■主内存
参数、技能卡、P道具将被反映。

■子存储器
部分参数和技能卡将被反映。主内存与計画類型不同的内存无法编組。
※不可重復的技能卡即使包含在多個記憶体中，也仅会反映一張。此外若包含已強化的技能卡，则优先采用已強化版本。

■参数
主内存参数和部分子内存参数将被反映。

■技能卡
编成的主/副記憶技能卡将被编入。
除了記憶持有的技能卡之外，根据主記憶的推荐效果，基本技能卡将添加到持有技能卡中。

■P道具
主内存的P项目将被编組。
